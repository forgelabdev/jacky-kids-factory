# Universe Bible

This folder contains the character and style definitions for the kids universe.
All 4 engines (YouTube, KDP, Etsy, POD) must use these same characters and style.

## What goes here

- `characters.md` — Character descriptions, names, personalities, colors
- `style_guide.md` — Visual style rules (line weight, color palette, proportions)
- `sd_prompts.md` — Stable Diffusion prompts that reproduce each character consistently
- `reference_images/` — Generated reference sheets for each character (PNG)

## Workflow

1. Define characters in `characters.md`
2. Generate reference images with SD using prompts from `sd_prompts.md`
3. Use the same prompts across ALL production folders to maintain consistency
