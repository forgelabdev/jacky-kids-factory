# POD Kids Designs

T-shirts, stickers, posters for kids — same characters as all other engines.

## Pipeline

```
Character (universe/) → SD design (ComfyUI) → RemBG (transparent) → Upscale (Real-ESRGAN) → Upload multi-platform
```

## Platforms

1. Redbubble — immediate, ~20-30% royalty
2. TeePublic — immediate, ~$4/t-shirt
3. Spring — immediate, custom margins
4. Merch by Amazon — apply Day 1, wait 2-8 weeks

## File naming convention

```
design_dino_learning_001.png     — Transparent PNG, 4500x5400 min
design_dino_learning_001.md      — Title, tags, description per platform
```

## Product focus

- Kids t-shirts (same characters)
- Stickers (high margin, impulse buys)
- Room posters/wall art
- Lunch bags, tablet cases

## Rules

- ALWAYS check trademarks before publishing (USPTO TESS, EUIPO TMview)
- NEVER use Disney, Peppa Pig, Pokemon, etc.
- PNG transparent, 300 DPI, CMYK for print
