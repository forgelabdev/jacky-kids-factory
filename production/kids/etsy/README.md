# Etsy Kids Digital Products

Worksheets, coloring pages, party decorations, reward charts.

## Pipeline

```
Theme (universe/) → SD images + Claude text → PDF/PNG assembly (Canva/Python) → Upload Etsy
```

## File naming convention

```
listing_coloring_dinosaurs_10pack.md    — Listing copy (title, description, 13 tags, price)
assets/coloring_dinosaurs/              — Product files (PDF + PNG)
```

## Product types (by priority)

1. Coloring pages (packs of 10-50)
2. Educational worksheets (numbers, letters, matching)
3. Activity bundles (coloring + worksheets + games)
4. Party decorations (birthday themes)
5. Reward/chore charts
6. Kids room wall art

## Seasonal calendar

- Apr: Easter worksheets
- May: Mother's Day cards
- Jul: Back-to-school prep
- Oct: Halloween coloring + activities
- Dec: Christmas activity packs (+800% sales)

## Pricing

- Coloring packs: $3-8
- Worksheet bundles: $4-10
- Party decorations: $5-15
- Activity mega-bundles: $8-20
