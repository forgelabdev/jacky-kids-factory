# YouTube Kids Production

Scripts and assets for the kids YouTube channel(s).

## Pipeline

```
Script (Claude) → Voice (ElevenLabs/XTTS) → Music (Suno AI) → Images (SD) → Assembly (CapCut/Animaker) → Upload
```

## File naming convention

```
script_EN_counting_01.md      — English counting video #1
script_EN_alphabet_02.md      — English alphabet video #2
script_FR_couleurs_01.md      — French colors video #1
```

## Content types (by priority)

1. Nursery rhymes compilations (20-40 min)
2. Counting/numbers songs
3. Alphabet & phonics
4. Colors & shapes
5. Animal sounds
6. Bedtime lullabies

## Rules

- Always mark "Made for Kids" (COPPA)
- Declare AI usage in description
- Same characters from `universe/` folder
