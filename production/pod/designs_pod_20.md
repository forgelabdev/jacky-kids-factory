# WORKFLOW DE PRODUCTION

## Étape par étape :

1. **Générer l'image** dans ComfyUI avec le prompt
   - Résolution : 1024x1024 minimum (SDXL)
   - Pour POD t-shirts : fond transparent (utiliser le modèle avec alpha channel)
   - Pour posters/wall art : fond opaque

2. **Upscaler** avec Real-ESRGAN
   - T-shirts : minimum 4500x5400 pixels (Redbubble)
   - Posters : minimum 6000x8000 pixels
   - `python -m realesrgan -i input.png -o output.png -s 4`

3. **Ajouter le texte** dans Canva
   - Police clean et lisible
   - Tester la lisibilité en petit (preview sticker)

4. **Upload multi-plateforme**
   - Redbubble : activer TOUS les produits
   - TeePublic : t-shirt + sticker + mug minimum
   - Spring : focus t-shirts et hoodies

## Tags par design (Redbubble) :
Utiliser 15 tags max. Inclure :
- Le sujet principal (ex: "nurse", "cat", "coding")
- Le style (ex: "funny", "minimalist", "vintage")
- L'occasion (ex: "gift for nurse", "christmas gift")
- Les synonymes (ex: "programmer", "developer", "coder")
